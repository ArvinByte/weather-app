# Weather App â›…

A simple desktop application built with **Python** and **Tkinter** that shows the current weather of any city using the **OpenWeatherMap API**.

---

## Features
- Get real-time weather conditions
- Display temperature in Celsius
- Weather condition description
- Error messages for invalid inputs
- Clean and simple UI

---

## Technologies Used
| Technology | Purpose |
|-----------|---------|
| Python | Main programming language |
| Tkinter | Graphical user interface |
| Requests | API communication |
| OpenWeather API | Weather data source |

---

## Requirements (Installation)
Make sure you have Python installed.

Install required library:
```bash
pip install requests
```
---
## How to Run
```bash
python weather_app.py
```
Enter the name of a city in the text box and click Get Weather.
---
## API Setup

This project uses OpenWeatherMap.

Get your free API Key here:
https://home.openweathermap.org/api_keys

Replace your API key inside the script:
```bash
API_KEY = "YOUR_API_KEY"
```
